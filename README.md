# Tetris

A browser-based Tetris recreation in a single-file HTML app.

## Overview

This repository contains a standalone `tetris.html` file that runs directly in a modern browser with no build tools or external dependencies.

The game includes:

- Standard 10×20 playfield with all 7 tetrominoes
- 7-bag piece randomizer
- Left/right movement, soft drop, hard drop, and rotation
- Basic wall-kick behavior
- Ghost piece preview
- Next-piece preview
- Line clearing with score multipliers
- Level progression and increasing drop speed
- Score, lines, and level display
- Pause/resume and reset controls
- Game-over detection
- Theme switcher with multiple visual styles
- Synthesized music and sound effects using the Web Audio API

## Controls

- `←` / `→`: move piece left/right
- `↑`: rotate piece
- `↓`: soft drop
- `Space`: hard drop
- `P`: pause/resume
- `R`: reset game
- `M`: mute/unmute audio

## Themes

The game supports multiple selectable themes, including:

- ASCII Terminal
- Game Boy
- Classic (default)
- Mono
- Cyberpunk 2077
- Glass
- Stone
- Bronze
- Gold
- Girly
- Super Mario Bros

## Usage

Open `tetris.html` in your browser to play.
